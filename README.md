# My Stuff

A premium personal entertainment tracker designed to help you organize movies, shows, anime, and more. Built with a focus on aesthetics, smooth animations, and a rich user experience.

[View on GitHub](https://github.com/Vamshikrishna779/My-Stuff)

## Features

- **Personalized Tracking**: Keep track of what you've watched and what you want to watch.
- **Premium Theming**: Choose from over 10 professionally curated themes (Stranger Things, Dracula, Cyberpunk, and more).
- **Dynamic Interface**: Smooth animations ("Jelly" opening effect), glassmorphism UI, and responsive design.
- **Smart Recommendations**: Get suggestions based on your collection using TMDB data.
- **Cast & Crew**: Detailed cast information with direct search integration.
- **PWA Support**: Installable on mobile and desktop with offline capabilities.
- **Cloud Sync**: seamless data synchronization across devices (powered by Firebase).
- **Admin Dashboard**: Analytics and user management features.

## Tech Stack

- **Frontend**: React, TypeScript, Vite
- **Styling**: Vanilla CSS (Variables & Glassmorphism), Framer Motion
- **Backend / specific services**: Firebase (Auth & Realtime Database), TMDB API
- **Deployment**: Vercel

## Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/Vamshikrishna779/My-Stuff.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
